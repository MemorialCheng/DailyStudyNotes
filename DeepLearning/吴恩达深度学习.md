# 4 CNN卷积神经网络
## 二维input
### 场景1：步长=1

__假设input为n*n，pading填充为 p，filter为f*f，则将output为(n + 2p - f + 1) * (n + 2p - f + 1)__

如果想要input与output相同size,则p=(f-1)/2. It`s actually almost always odd.

### 场景2：步长stride>1

公式：__[(n + 2p - f)/s + 1] * [(n + 2p - f)/s +1] __

## 三维input

### 存在多个filter
Nf表示filters个数

公式：__ [(n + 2p - f)/s + 1] * [(n + 2p - f)/s +1] * Nf__
